Identity Generator - A lightweight and fast console application written in Go that allows you to generate random user identities.
Features: Bilingual Generation: There are Russian and English versions, and the data is generated differently for each. For the Russian language, it creates a Last Name, First Name, and Middle Name, while for English, it generates only a First Name and Last Name.Gender 
Selection: You can manually choose the gender (Male / Female) or select Random Gender to let the system choose it for you.
Fast & Efficient: Powered by Go's native performance.
Where to Run: Since this is standard Go (Golang) code, you can run it absolutely anywhere: 
Terminal / Command Prompt: Directly via go run main.go on Windows, macOS, or Linux.IDEs: VS Code (with Go extension), GoLand, or Fleet.Text Editors: Vim, NeoVim, Sublime Text, or Notepad++ (using the terminal).
Docker: Inside any Linux container.
